# Leon Design System - Web Template

## Overview
Leon is a super creative and minimal agency web template built using HTML and CSS. It's designed to be a clean, responsive, and beautiful landing page for agencies or personal portfolios. This project represents a "First Design Using HTML & CSS" and demonstrates fundamental web development skills, layout structuring, and styling.

## Features
- **Responsive Design:** Optimized for different screen sizes (mobile, tablet, and desktop).
- **Smooth Scrolling Navigation:** Same-page navigation to easily jump to different sections (Services, About, Portfolio, Contact).
- **Clean UI/UX:** Minimalist and modern aesthetics.
- **FontAwesome Icons:** Integration of high-quality vector icons for visual enhancement.
- **Normalize.css:** Ensures consistent styling across different browsers.

## Sections Included
- **Hero Section:** A welcoming introduction area.
- **Features:** Highlights key offerings ("Tell Us Your Idea", "We Will Do All The Work", etc.).
- **Services:** Details specific skills like Graphic Design, UI & UX, Web Design, and Web Development.
- **Portfolio:** A showcase of past projects with images and descriptions.
- **About:** Information about the creator/agency, background, and skills.
- **Contact:** Provides a way for users to get in touch via email and social media links.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS)
- FontAwesome (for icons)
- Normalize.css (for browser consistency)

## How to Run
Simply open the `index.html` file in any modern web browser to view the template. No build process or server is required.
